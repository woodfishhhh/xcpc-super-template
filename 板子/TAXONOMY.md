# 公共模板分类与学习顺序

公共模板按 `板子/<顶级分类>/<二级分类>/meta.json + code.md` 组织。`meta.json` 里的 `category` 必须和目录路径一致，`learningOrder` 必须落在对应顶级分类的范围内。

| 顶级分类 | learningOrder 范围 | 内容 |
| --- | ---: | --- |
| 基础算法 | 1-99 | 排序、二分、搜索、贪心等入门基础 |
| 数据结构 | 100-199 | 并查集、树状数组、线段树、RMQ 等常用结构 |
| 图论 | 200-299 | 最短路、生成树、连通性、拓扑序、匹配等图算法 |
| 字符串 | 300-399 | 字符串匹配、哈希、自动机、后缀结构等 |
| 数学与数论 | 400-499 | gcd、逆元、组合数、筛法、矩阵等 |
| 动态规划 | 500-599 | 背包、区间、树形、数位、状态压缩等 |
| 几何 | 600-699 | 二维几何、凸包、半平面、最近点对等 |
| 网络流 | 700-799 | 最大流、最小割、费用流、匹配建模等 |
| 杂项 | 900-999 | 随机化、调试、输入输出、工程辅助模板 |

新增模板前先选定顶级分类，再在该分类范围内给出相对学习顺序。范围中间预留空位，方便后续插入更基础或更进阶的模板。
